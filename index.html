Hex codes

Hexadecimal color codes are written in the format #RRGGBB and represent levels of red, green, and blue.

Each of the six characters can be any combination of 0-9 and A-F (00 is the lowest; FF is the highest).

There are 16,777,216 possible combinations/colors.

#r {background-color: #ff0000;}    /*red*/
#g {background-color: #00ff00;}    /*green*/
#b {background-color: #0000ff;}    /*blue*/

RGB values

The syntax to set colors using an RGB value is rgb(red,green,blue).

Three numbers between 0 and 255, or a percentage from 0% to 100%, specify the intensity of red, green, and blue.


Black
0,
0,0


Gray
128,128,
128


Light­Gray
211,
211,211


White
255,255,
255

Black is the "absense of color", which might be why rgb(0,0,0) = black.

To make gray, set all 3 values to the same number.
Numbers closer to 255 will be lighter.

White, the lightest possible color, is made when all 3 numbers are set at the max, 255.
