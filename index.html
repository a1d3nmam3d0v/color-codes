External CSS

External style sheets (.CSS files) are connected with a <link> element in the <head> of the HTML 

<!DOCTYPE html>
<html lang="en">
    <head>
        <link rel="stylesheet" href="exampleStyleSheet.CSS" />
    </head>
    <body>
        <h1>Heading</h1>
            <p>The external css file is linked above</p>
    </body>
</html>

Inline CSS

<p style="color:darkorange;" > Styles can be applied directly in the HTML by adding a "style" attribute—containing any CSS property—to an element. </p>

Color names

Colors can be set with predefined names.
Modern browsers support 140 different color keywords. 

p {
  color: green;          /* this will make your text green */
 }
And it will look like this 
p {  
  color: blue;              /* blue text */
  background-color: pink;   /* pink background */
 }
this is blue text with a pink background

Hex codes

Hexadecimal color codes are written in #RRGGBB format, representing levels of red, green, and blue.
Each of the six character can be any combination of 0-9 and A-F (lowest is 00; highest is FF).
16,777,216 possible combinations/colors. 

  #r {background-color: #ff0000;}        /*red*/
  #g {background-color: #00ff00;}       /*green*/
  #b {background-color: #0000ff;}       /*blue*/

RGB values

The syntax to set colors in CSS using an RGB value is rgb(red,green,blue)
Three numbers between 0 and 255, or a percentage from 0% to 100%, specify the intensity of red, green, and blue.

black
0 0 0
dim­gray
105 105 105
gray
128 128 128
dark­gray
169 169 169
light­gray
211 211 211
white
255 255 255

Black is the "absense of color" therefore rgb(0,0,0) = black.
To make gray, set all 3 values to the same number. Numbers closer to 0 are darker.
White is the lightest possible color and is made when all 3 numbers are set at the max, 255.
